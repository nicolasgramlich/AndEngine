Detailed Description

Note: ButtonSprite needs to be registered as a ITouchArea to the Scene via Scene#registerTouchArea(ITouchArea), otherwise it won't be clickable. To make ButtonSprite function properly, you should consider setting Scene#setTouchAreaBindingOnActionDownEnabled(boolean) to true.
